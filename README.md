## ðŸ”¬ Replication Guide for Candidate Generation (Graph Models)

The input for `LLM_validation1125run.ipynb` is a list of candidate drugs. These candidates were generated using three state-of-the-art models. To replicate this upstream step, we utilized the official implementations adapted for the **PrimeKG** dataset:

### 1. TxGNN
We utilized the official pre-trained weights and zero-shot prediction scores.
* **Repository:** [https://github.com/mims-harvard/TxGNN](https://github.com/mims-harvard/TxGNN)

### 2. CompGCN
We retrained the model on PrimeKG using the standard link prediction setting.
* **Repository:** [https://github.com/malllabiisc/CompGCN](https://github.com/malllabiisc/CompGCN)

### 3. RLR (Hetionet)
We computed Degree-Weighted Path Counts (DWPC) using the `learn` and `hetnet_ml` packages.
* **Repositories:**
    * [https://github.com/dhimmel/learn](https://github.com/dhimmel/learn)
    * [https://github.com/mmayers12/hetnet_ml](https://github.com/mmayers12/hetnet_ml)
